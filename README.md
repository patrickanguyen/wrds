# WRDS

A Radio Data System (RDS) decoder library written in Rust.

> [!WARNING]
> This library is very experimental and should not be used for anything serious.

## Supported fields
- [X] Programme Identifier (PI)
- [X] Programme Type (PTY)
- [X] Traffic Program (TP)
- [ ] Radio Text (RT)
- [ ] Radio Text Plus (RT+)
- [ ] Everything else
